🚀 About the Project
Instant Fix is an AI-powered on-demand home services platform that connects users with verified local professionals — electricians, plumbers, car mechanics, and more — in minutes.
Starting in Lucknow, India, Instant Fix aims to solve the fragmented, unreliable, and untrustworthy home repair market by bringing transparency, speed, and intelligence to everyday service needs.
